# تطبيق جزء عم - المصحف المعلم للطفل
## Juz Amma - Teacher Quran for Kids

تطبيق إسلامي تعليمي مخصص للأطفال، يهدف إلى تسهيل حفظ وترديد جزء عمّ بصوت الشيخ **محمد شريف**. يتميز التطبيق بتصميم عصري (Premium UI) وتجربة مستخدم سلسة تدعم التعلم التفاعلي.

**تطوير وإعداد: [عبدالرحمن غنيم](https://github.com/abdulrahmanghoneam-debug)**
**Developed by: Abdulrahman Ghonim**

---

## مميزات التطبيق (Features)

- **المصحف المعلم**: تكرار الآيات خلف الشيخ لتسهيل الحفظ.
- **عرض الآيات**: مزامنة عرض نص الآيات مع الصوت بدقة عالية.
- **دعم الأوفلاين (PWA)**: إمكانية تحميل السور والتعامل مع التطبيق بدون إنترنت.
- **واجهة مستخدم متميزة**: تصميم "Glassmorphic" مع حركات وأنيميشن سلسة.
- **نظام المفضلات**: حفظ السور المفضلة للوصول السريع.

## التقنيات المستخدمة (Tech Stack)

- **React + Vite**
- **TypeScript**
- **Tailwind CSS + Shadcn UI**
- **Framer Motion** (للحركات والأنيميشن)
- **Capacitor** (لتحويله إلى تطبيق mobile)
- **Supabase** (للبنية التحتية)

## طريقة التشغيل محلياً

```sh
# تثبيت التبعيات
npm install

# تشغيل خادم التطوير
npm run dev
```
